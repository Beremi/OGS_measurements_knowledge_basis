# CD-A OGS Blocker Discussion Packet

This folder contains a shareable LaTeX packet that consolidates the current
external-provider questions, request-email review state, response instructions,
and modelling-team decisions blocking final all-measurement field promotion.
The PDF is self-contained and does not rely on local folders or tracker files.

- Source: `blocker_discussion_packet.tex`
- PDF: `blocker_discussion_packet.pdf`

The packet documents request emails and provider questions only. It does not send,
archive, delete, label, or otherwise modify any email.

Build from this folder with:

```bash
pdflatex -interaction=nonstopmode blocker_discussion_packet.tex
pdflatex -interaction=nonstopmode blocker_discussion_packet.tex
```

The `.tex` file is the editable source for the standalone PDF.
