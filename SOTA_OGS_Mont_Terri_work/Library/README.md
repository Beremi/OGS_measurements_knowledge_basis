# Library

This folder tracks literature and source material used while revising the CD-A / OGS report.

## Fulltexts

Downloaded or locally copied fulltexts are in [fulltexts](fulltexts).

Current fulltext set:

- `Archie_1942_Electrical_Resistivity_Log.pdf` - primary source for the empirical resistivity/saturation relation.
- `Amann_et_al_2017_Geomechanical_Opalinus_Clay.pdf` - geomechanical and petrophysical behaviour of Opalinus Clay at Mont Terri.
- `Garitte_et_al_2017_HE_D_THM_Opalinus_Clay.pdf` - DECOVALEX HE-D THM modelling and anisotropy/heating-test context.
- `Gautschi_2017_Safety_Relevant_Hydrogeological_Opalinus_Clay.pdf` - hydrogeological property ranges and multiple-lines-of-evidence context for Opalinus Clay.
- `Gens_et_al_2017_Thermal_Loading_Opalinus_Clay.pdf` - Mont Terri heating-test observations and THM response.
- `Gens_Vaunat_Garitte_Wileveau_2007_In_situ_behaviour_OPA.pdf` - open-access PDF of the Geotechnique heating-test interpretation paper, downloaded from the UPC repository.
- `Input_HERMES_BGR_20241217.pdf` - local HERMES input note from the email knowledge base.
- `Klinkenberg_1941_Permeability_Liquids_Gases.pdf` - primary source for gas-slip correction and the distinction between gas-measured apparent permeability and liquid/intrinsic permeability.
- `Local_CD_A_for_HERMES_2D_250904x.pdf` - local September 2025 CD-A modelling slides used for ERT, permeability-test, Taupe, and projection-workflow details.
- `Local_BGR_Modelling_TD_CDA_2026.pdf` - local 2026 BGR modelling slides used for other-HM validation-priority and extensometer caveat statements.
- `Local_NMR2026_CD_A.pdf` - local NMR 2026 slide deck used for the interlayer/bound-water caveat.
- `Local_Precision_Levelling_TD_CDA_2026.pdf` - local 2026 precision-levelling slides used for pointwise vertical displacement summaries.
- `Local_Taupe_TD_CD_A_ISU_2026.pdf` - local Taupe/TDR slide deck used for the differential-TDR method statement.
- `Local_TD517_CD_A_260507_Minutes.pdf` - local May 2026 technical-discussion minutes used for RH/suction sensor-reliability statements.
- `Marschall_Horseman_Gimmi_2005_Gas_Transport_Opalinus_Clay.pdf` - gas transport/permeability evidence for Opalinus Clay.
- `Revil_et_al_1998_Electrical_Conductivity_Shaly_Sands.pdf` - JGR fulltext used to support the ERT clay/surface-conduction caveat.
- `Robinson_et_al_2003_TDR_Dielectric_EC_Review_USDA.pdf` - USDA-hosted fulltext used to support the Taupe/TDR dielectric-water-content calibration caveat.
- `Cui_Shikhov_Arns_2022_NMR_Relaxation_Dual_Scale.pdf` - open-access Springer fulltext used to support the NMR clay/micro-porosity forward-modelling caveat.
- `Thomson_1871_Equilibrium_Vapour_Curved_Surface.pdf` - Zenodo-hosted copy of the primary historical Kelvin-equation source.
- `Wang_Kosakowski_Kolditz_2009_THM_OGS.pdf` - OGS THM reference bundled in the report archive.
- `Van_Loon_2004_Nagra_NTB_03_07_Diffusion_OPA.pdf` - Nagra NTB 03-07 report on HTO, chloride, iodide, and sodium diffusion in Mont Terri Opalinus Clay, used for porosity/rock-capacity context.
- `Villar_Romero_2012_Water_Retention_Curves_Opalinus_Clay.pdf` - CIEMAT technical report on Opalinus Clay retention curves and air-entry values.
- `Ziefle_et_al_2021_CD_A_setup_first_results.pdf` - open-access CD-A setup / first-results conference abstract.
- `Ziefle_et_al_2024_CDA_twin_niches_characterization.pdf` - local fulltext from the email archive.
- `Ziefle_et_al_2024_Water_Content_EDZ_Opalinus_Clay.pdf` - open-access Springer fulltext.
- `van_Genuchten_1980_Hydraulic_Conductivity_Unsaturated_Soils.pdf` - USDA-hosted fulltext copy of the van Genuchten retention/relative-conductivity paper.

HTML files with `_download_blocked` or `_download_page` suffix are failed fulltext-download attempts retained as provenance. Some have since been superseded by clean PDFs listed above.
