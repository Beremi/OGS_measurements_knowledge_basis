# CD-A OGS TRM Report: Overleaf Package

This folder is self-contained for Overleaf.

- Root file: `main.tex`
- Bibliography: `references.bib`
- Figures: `figures/`
- Recommended compiler: pdfLaTeX
- Bibliography tool: BibTeX

On Overleaf, upload the contents of this folder and set `main.tex` as the main
document.  The project does not require local scripts, local data folders, or
external generated files to compile.
